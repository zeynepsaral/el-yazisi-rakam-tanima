# El Yazısı ile Rakam Tanıma

Bu projede, MNIST veri seti kullanılarak el yazısı ile yazılmış
rakamların PyTorch yardımıyla sınıflandırılması amaçlanmıştır.

Çalışma kapsamında temel bir CNN modeli kurulmuş ve
modelin farklı rakamlar üzerindeki başarımı incelenmiştir.

## Kullanılan Teknolojiler
- Python
- PyTorch
- NumPy
- Matplotlib

## Proje Yapısı
- kaynak/: modelin eğitimi ve testine ait kodlar
- benim rakamlarım/: kullanıcı tarafından çizilen rakam örnekleri
- eserler/: eğitim sürecinde elde edilen çıktılar

## Amaç
Derin öğrenme ve evrişimsel sinir ağları (CNN) konusunda
uygulamalı deneyim kazanmak ve
el yazısı rakam tanıma problemini anlamak.
